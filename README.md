# General Architecture
- Stack: React (Typescript) only
- Database: Oracle Autonomous JSON database v19c

# How to Run
## Prerequisites
- Node.js v18
- NPM v9
- Yarn v1.22

## Steps
1. To start the app locally, run 
```bash
yarn install
yarn start
```
2. The app will be available at `http://localhost:3000`
